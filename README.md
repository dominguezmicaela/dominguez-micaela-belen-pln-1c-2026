# Técnicas de Procesamiento del Habla — PLN 1C 2026

Repositorio de proyectos y ejercicios prácticos desarrollados a lo largo de la materia **Técnicas de Procesamiento del Habla**, cursada en 2026.

## Descripción

Este repositorio reúne todos los trabajos, notebooks y resolucion de ejercicios asociados a la materia. 

## Tecnologías utilizadas

- Python 3.13
- Jupyter Notebooks
- Librerías de PLN (Natural Language Processing)

## Cómo ejecutar

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/dominguez-micaela-belen/dominguez-micaela-belen-pln-1c-2026.git
   ```

2. Crear y activar el entorno virtual:
   ```bash
   python -m venv venv
   venv\Scripts\Activate.ps1   # Windows
   ```

3. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
   ```

4. Abrir los notebooks con Jupyter o VS Code.
